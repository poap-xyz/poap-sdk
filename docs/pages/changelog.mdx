# Changelog

## 0.17.0

**Features**

- [Drops] Add dedicated `DropsClient.get` method to get a single drop by ID.

**Breaking Changes ⚠️**

- [Drops] `DropsClient.fetch` has been renamed to `DropsClient.list`.

## 0.16.0

**Features**

- [Collections] Add Collections module. See [Collections docs](https://sdk.poap.tech/modules/collections/GetCollection).

## 0.15.0

**Maintenance**

- Upgrade to Node.js LTS (22).
- Upgrade GitHub Action checkout to v5.
- Migrate from `yarn` to `pnpm`.

## 0.14.0

**Features**

- [POAPs] Add `PoapsClient.fetchCount` method to get the total number of POAPs matching given filters.

## 0.13.0

**Breaking Changes ⚠️**

- Subpath imports such as `@poap-xyz/poap-sdk/utils` are no longer supported. Use `@poap-xyz/poap-sdk` instead.

**Maintenance**

- Remove multi-package build configuration and file architecture. The SDK is now built as a single package.
- Remove `@poap-xyz/frames` package (still published standalone on npm, but no longer part of SDK monorepo).

## 0.12.0

**Features**

- [Profiles] Add errors in the response of bulk profile fetching (supports retry strategy).

**Breaking Changes ⚠️**

- [Profiles] `ProfilesClient.fetchBulk` now returns an object with `profiles` and `errors` maps.

**Maintenance**

- Upgrade to yarn 4.
- [Moments] Use `Uint8Array` instead of `Buffer` for Moments media upload and remove node polyfill.

## 0.11.0

**Breaking Changes ⚠️**

- [Drops] `DropsClient.fetch` property changes:
  - `name` filter removed (use `DropsClient.search` instead).
  - `isPrivate` filter removed.
  - `from`/`to` now filter by drop **event** dates instead of creation date.

## 0.10.6

**Features**

- Add `nulls_first` and `nulls_last` options to the `Order` enum.

## 0.10.5

**Features**

- [Profiles] Add serializable profile type and related methods to make a profile response serializable and vice versa.

## 0.10.4

**Features**

- [Profiles] Keep only 1 profile when multiple are returned for the same query.

## 0.10.3

**Features**

- [Drops] Add HTTP request options to `DropsClient.fetch`.

## 0.10.2

**Features**

- [Profiles] Reduce Profiles API bulk request limit from 100 to 50.

## 0.10.1

**Features**

- [Profiles] Add HTTP request options to `ProfilesClient.fetch`.

## 0.10.0

**Features**

- [Profiles] Add Profiles API client. See [Profiles docs](https://sdk.poap.tech/modules/profiles/FetchProfile).
